# realtime-isl-recognition
Real-time isl(isolated sign language) recognition project

# data_preprocess

Normalizes raw landmark CSV files (hand + body pose) so they are ready for model training.

---

## Dataset

Due to GitHub file size limits, the full dataset is hosted externally.

Download here: [Google Drive Link]
